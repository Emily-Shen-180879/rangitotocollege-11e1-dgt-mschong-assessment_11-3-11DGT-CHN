Exercises and Tutorials

Here are some exercises and tutorials for more practice and a better understanding of the topic:

Shapes

Drawing shapes is a good way to improve your understanding on 3D space and form. When drawing shapes, try to not focus on making it perfect, but rather focus on drawing as many shapes as you can think of.

Anatomy

Once you've gotten some practice by drawing some 3D shapes, you can begin to learn anatomy using the skills you learned from practicing drawing shapes.
